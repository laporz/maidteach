キャラ画像ファイル命名規則
===========================

形式: chara_{身長}_{体型}_{服}_{髪型}_{髪色}.png

身長:   low（低身長） / high（高身長）
体型:   slender（スレンダー） / glamorous（グラマラス）
服:     maid_mini / maid_long / sailor_white / sailor_black / virgin_killer
        negligee_white / negligee_black / bikini_black / micro_white / bandage
髪型:   long / short / semi_long / twin / pony
髪色:   gold / white / black / red / pink / green / blue

例:
  chara_low_slender_maid_mini_long_gold.png  （初期状態）
  chara_high_glamorous_sailor_white_twin_pink.png
  chara_low_slender_negligee_white_long_gold.png

画像が存在しない場合は「（画像なし）」と表示されます。
